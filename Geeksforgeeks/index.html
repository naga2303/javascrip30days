String Methods

charAt - characterIndex charcodeAt - returns unicode value concat - combines two strings and return a new string. indexof - return the index value of the first identified String lastIndexof - returns the index value of the last occurrence. match - used to match a regular expression against a string. replace - used to find a match and replace the string. search - executes the search for a match between a regular expression and the string. slice - extracts a section of a string and returns new string. substring() - It is used to fetch the part of the given string on the basis of the specified index. toLowercase - converts the string into toLowercase toUpperCase - converts the string into toUpperCase split() - splits a string into substring array, then returns newly created array. trim - removes whitespaces from left and right side of the string.

Array methods

concat - It return a new array that contains two or more merged arrays. entries() - It creates a iterator object and a loop which traverses to key/value pair. every() - It determines whether all the elements of the array satisfies the provided condition. from() - It creates a new array carrying the exact copy of another array element. filter() - It returns the new array containing the elements that pass the provided function condition. find() - It return the first element of the give array that satisfies the condition. findIndex() - It returns the index value of the first element of the given array that satisfies the condition. forEach() - It invokes the function once for each value of the array. includes() - It checks in the given array whether the specified element contains. indexof() - It searches the specified element in the given array and returns the index of the first match. join() - It joins the elements of the array as a string. pop() - It removes and returns the last element of the array. push() - It add the element in the last index of the array. shift() - It removes and returns the first element of the array. slice() - it return a new array containing the copy of that array. splice() - It adds/removes elements to/from the given array. Unshift() - It adds the element at the start of the given array.
